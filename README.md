# BIM
Working towards Digital Transformation in Civil Engineering, using  Open Standards

## Open Source Softwares On Which We Work
* [FreeCAD](https://www.freecad.org)
Role: Parametric 3D Modeling & BIM Authoring.

Description: A multidisciplinary parametric 3D modeler. While similar to AutoCAD 3D in output, it excels in BIM-specific workflows through its dedicated BIM Workbench and Native IFC support.

* [OpenSCAD](https://openscad.org)
Role: Functional & Algorithmic Design.

Description: Unlike traditional CAD, OpenSCAD is a compiler that renders 3D models based on script files. It allows for absolute precision in structural components using code-based logic and classes
